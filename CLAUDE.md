# CodeRabbit Issue Fixer Instructions

You are fixing a GitHub issue created by the CodeRabbit bot. Follow these rules strictly:

## Rules

1. **Read the issue carefully** — understand what the issue is about before making changes
2. **Make minimal, focused changes** — only fix what the issue describes, nothing else
3. **Do not modify unrelated code** — no refactoring, no style changes outside the fix scope
4. **Follow existing conventions** — read and follow AGENTS.md if it exists in the repo
5. **Run checks after changes**:
   - `pnpm lint:fix` to fix any lint issues
   - `pnpm typecheck` to verify type safety
6. **Stage your changes** — use `git add` on the files you modified
7. **Do not commit** — the orchestration script handles committing
